# EmoTales AI: Full-Stack Educational Platform

EmoTales AI is an innovative story generation platform designed for educational purposes. It uses a modern tech stack to provide a seamless experience for generating and managing interactive stories.

##  Tech Stack
* **Frontend:** React.js (Tailwind CSS)
* **Backend:** FastAPI (Python)
* **Database:** PostgreSQL
* **Containerization:** Docker & Docker Compose

##  Features
* AI-powered story generation.
* User-friendly dashboard for story management.
* Full containerization for easy deployment.

##  How to Run with Docker

To get the entire system up and running, ensure you have **Docker Desktop** installed, then run the following command in the root directory:

```bash
docker-compose up --build
